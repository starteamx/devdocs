# Python 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 Python 语言的基本语法和核心概念。

- **变量与数据类型**：基本类型、容器类型、类型转换
- **控制流**：条件语句、循环、异常处理
- **函数**：函数定义、参数传递、作用域
- **数据结构**：列表、字典、集合、元组
- **面向对象编程**：类与对象、继承、多态
- **模块与包**：模块导入、包管理、标准库

[📚 深入学习语法基础 →](https://docs.python.org/zh-cn/3/tutorial/index.html)

## Phase 2: 核心工具

掌握 Python 标准库中的核心工具和常用框架。

- **文件操作**：文件读写、路径处理、CSV/JSON 处理
- **正则表达式**：模式匹配、文本处理
- **日期时间**：datetime、timezone 处理
- **网络编程**：socket、HTTP 客户端、requests
- **数据库操作**：SQLite、MySQL、PostgreSQL 连接
- **并发编程**：多线程、多进程、异步编程

[📚 深入学习核心工具 →](https://docs.python.org/zh-cn/3/tutorial/index.html)

## Phase 3: 高级特性

掌握 Python 的高级编程特性和现代编程范式。

- **装饰器**：函数装饰器、类装饰器、应用场景
- **生成器与迭代器**：yield、生成器表达式、迭代器协议
- **上下文管理器**：with 语句、自定义上下文管理器
- **元编程**：元类、描述符、属性访问
- **类型注解**：类型提示、类型检查、mypy
- **异步编程**：asyncio、协程、异步IO

[📚 深入学习高级特性 →](https://docs.python.org/zh-cn/3/tutorial/index.html)

## Phase 4: 底层原理

深入理解 Python 运行机制和性能优化。

- **Python 解释器**：CPython、PyPy、Jython、IronPython
- **内存管理**：引用计数、垃圾回收、内存优化
- **性能优化**：性能分析、C扩展、Cython
- **包管理**：pip、虚拟环境、conda、poetry
- **项目结构**：项目组织、打包分发、发布到 PyPI
- **最佳实践**：代码规范、测试、文档、CI/CD

[📚 深入学习底层原理 →](https://docs.python.org/zh-cn/3/tutorial/index.html)

---

💡 **提示**：每个阶段的学习内容都链接到 [Python 官方教程](https://docs.python.org/zh-cn/3/tutorial/index.html)，建议按照顺序系统学习。
