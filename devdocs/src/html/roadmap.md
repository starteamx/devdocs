# HTML 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 HTML 语言的基本语法和核心概念。

- **HTML 基础**：文档结构、DOCTYPE、html 元素
- **文本元素**：标题、段落、列表、引用
- **链接与图片**：a 标签、img 标签、路径
- **表格**：table、tr、td、th、表格属性
- **表单基础**：form、input、button、label
- **语义化标签**：header、nav、main、article、section

[📚 深入学习语法基础 →](https://developer.mozilla.org/en-US/docs/Learn/HTML)

## Phase 2: 核心工具

掌握 HTML 的核心功能和常用元素。

- **表单进阶**：select、textarea、checkbox、radio、表单验证
- **多媒体**：audio、video、source、track
- **嵌入内容**：iframe、embed、object、canvas
- **元数据**：meta 标签、SEO、Open Graph
- **HTML5 特性**：语义化标签、data 属性、自定义属性
- **可访问性**：ARIA 属性、无障碍访问、语义化

[📚 深入学习核心工具 →](https://developer.mozilla.org/en-US/docs/Learn/HTML)

## Phase 3: 高级特性

掌握 HTML 的高级功能和现代 Web 标准。

- **Web Components**：自定义元素、Shadow DOM、模板
- **表单 API**：FormData、表单验证 API、约束验证
- **媒体 API**：Media API、Canvas API、WebGL
- **存储 API**：localStorage、sessionStorage、IndexedDB
- **Web Workers**：Worker、SharedWorker、消息传递
- **Service Worker**：PWA、离线支持、缓存策略

[📚 深入学习高级特性 →](https://developer.mozilla.org/en-US/docs/Learn/HTML)

## Phase 4: 底层原理

深入理解 HTML 标准和 Web 技术。

- **HTML 标准**：HTML5 规范、WHATWG、W3C
- **DOM 操作**：DOM 树、节点操作、事件模型
- **性能优化**：页面加载优化、资源优化、SEO
- **浏览器兼容性**：兼容性处理、polyfill、特性检测
- **最佳实践**：代码规范、语义化、可访问性
- **现代 Web**：Web 标准、渐进增强、响应式设计

[📚 深入学习底层原理 →](https://developer.mozilla.org/en-US/docs/Learn/HTML)

---

💡 **提示**：每个阶段的学习内容都链接到 [HTML 官方教程](https://developer.mozilla.org/en-US/docs/Learn/HTML)，建议按照顺序系统学习。
