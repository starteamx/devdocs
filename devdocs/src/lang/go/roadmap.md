# Go 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 Go 语言的基本语法和核心概念。

- **变量与数据类型**：基本类型、复合类型、类型转换
- **控制流**：if/else、switch、for 循环、range
- **函数**：函数定义、多返回值、可变参数
- **数组与切片**：数组、切片、切片操作
- **映射（Map）**：map 创建、操作、遍历
- **结构体**：结构体定义、方法、嵌入

[📚 深入学习语法基础 →](https://go.dev/doc/effective_go)

## Phase 2: 核心工具

掌握 Go 标准库中的核心工具和常用功能。

- **包管理**：包导入、模块系统、go mod
- **错误处理**：error 接口、错误处理模式、panic/recover
- **文件操作**：文件读写、路径处理、JSON/XML
- **字符串处理**：字符串操作、格式化、正则表达式
- **时间处理**：time 包、时区、定时器
- **并发基础**：goroutine、channel、select

[📚 深入学习核心工具 →](https://go.dev/doc/effective_go)

## Phase 3: 高级特性

掌握 Go 的高级编程特性和并发编程。

- **接口**：接口定义、接口实现、空接口
- **并发编程**：goroutine、channel、select、sync 包
- **反射**：reflect 包、类型反射、值反射
- **泛型**：类型参数、约束、泛型函数（Go 1.18+）
- **测试**：单元测试、基准测试、测试工具
- **上下文（Context）**：context 包、取消、超时

[📚 深入学习高级特性 →](https://go.dev/doc/effective_go)

## Phase 4: 底层原理

深入理解 Go 运行机制和性能优化。

- **Go 运行时**：Goroutine 调度、GC、内存管理
- **性能优化**：性能分析、pprof、优化技巧
- **编译与链接**：编译过程、交叉编译、构建标签
- **CGO**：C 语言互操作、FFI
- **项目结构**：项目组织、包设计、最佳实践
- **最佳实践**：代码规范、错误处理、并发模式

[📚 深入学习底层原理 →](https://go.dev/doc/effective_go)

---

💡 **提示**：每个阶段的学习内容都链接到 [Go 官方文档](https://go.dev/doc/effective_go)，建议按照顺序系统学习。
