# HTML

> TIOBE 排名 #11 | Web 的骨架。所有网页的起点，语义化的真理。

## 📘 概述

HTML 是 Web 的基础，定义了网页的结构和内容，是所有 Web 开发者的起点。

## 🔗 快速导航

- [📖 API 文档](https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element) - MDN HTML 元素参考
- [🎓 教程](https://developer.mozilla.org/zh-CN/docs/Learn/HTML) - MDN HTML 学习指南
- [🌐 标准规范](https://html.spec.whatwg.org/multipage/) - WHATWG HTML 标准

## 💻 代码示例

```html
<!-- HTML 的核心优势：语义化 + 可访问性 -->
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>语义化 HTML</title>
</head>
<body>
    <!-- 语义化标签 - 清晰的结构 -->
    <header>
        <nav>
            <ul>
                <li><a href="/">首页</a></li>
                <li><a href="/about">关于</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <h1>文章标题</h1>
            <section>
                <p>文章内容...</p>
            </section>
        </article>
    </main>
    
    <footer>
        <p>&copy; 2024 版权所有</p>
    </footer>
</body>
</html>
```

## 🎯 应用场景

- **Web 开发**：所有网页的基础
- **前端开发**：与 CSS、JavaScript 配合
- **SEO 优化**：语义化标签的重要性
- **可访问性**：无障碍访问支持
