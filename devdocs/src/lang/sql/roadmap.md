# SQL 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 SQL 语言的基本语法和核心概念。

- **数据库基础**：数据库、表、字段、数据类型
- **数据查询**：SELECT、FROM、WHERE、ORDER BY
- **数据操作**：INSERT、UPDATE、DELETE
- **条件查询**：比较运算符、逻辑运算符、IN、LIKE
- **排序与限制**：ORDER BY、LIMIT、OFFSET
- **聚合函数**：COUNT、SUM、AVG、MAX、MIN、GROUP BY

[📚 深入学习语法基础 →](https://www.w3schools.com/sql/)

## Phase 2: 核心工具

掌握 SQL 的核心功能和高级查询。

- **多表查询**：JOIN、INNER JOIN、LEFT JOIN、RIGHT JOIN
- **子查询**：子查询、EXISTS、IN、ANY、ALL
- **集合操作**：UNION、INTERSECT、EXCEPT
- **窗口函数**：ROW_NUMBER、RANK、DENSE_RANK、聚合窗口函数
- **索引**：索引创建、索引类型、索引优化
- **视图**：视图创建、视图使用、物化视图

[📚 深入学习核心工具 →](https://www.w3schools.com/sql/)

## Phase 3: 高级特性

掌握 SQL 的高级功能和数据库设计。

- **存储过程**：存储过程创建、参数传递、控制流
- **触发器**：触发器创建、触发时机、应用场景
- **事务处理**：事务、ACID、隔离级别、锁
- **数据库设计**：范式、ER 图、关系设计
- **性能优化**：查询优化、执行计划、索引优化
- **高级 SQL**：递归查询、CTE、JSON 处理

[📚 深入学习高级特性 →](https://www.w3schools.com/sql/)

## Phase 4: 底层原理

深入理解数据库运行机制和性能优化。

- **数据库引擎**：存储引擎、查询优化器、执行计划
- **索引原理**：B+树、哈希索引、全文索引
- **事务与锁**：MVCC、锁机制、死锁处理
- **性能调优**：慢查询分析、索引优化、分区
- **数据库管理**：备份恢复、监控、维护
- **最佳实践**：SQL 规范、安全、性能优化

[📚 深入学习底层原理 →](https://www.postgresql.org/docs/current/index.html)

---

💡 **提示**：每个阶段的学习内容都链接到 [SQL 学习教程](https://www.w3schools.com/sql/) 和 [PostgreSQL 文档](https://www.postgresql.org/docs/current/index.html)，建议按照顺序系统学习。
