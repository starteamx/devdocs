# JavaScript 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 JavaScript 语言的基本语法和核心概念。

- **变量与数据类型**：var/let/const、基本类型、对象类型
- **控制流**：条件语句、循环、switch
- **函数**：函数声明、函数表达式、箭头函数
- **对象与数组**：对象创建、数组操作、解构赋值
- **作用域与闭包**：作用域链、闭包、this 绑定
- **ES6+ 基础**：let/const、模板字符串、默认参数

[📚 深入学习语法基础 →](https://developer.mozilla.org/en-US/docs/Learn/JavaScript)

## Phase 2: 核心工具

掌握 JavaScript 标准库和常用 API。

- **数组方法**：map、filter、reduce、forEach、find
- **对象操作**：Object 方法、属性描述符、原型链
- **字符串处理**：字符串方法、正则表达式
- **日期时间**：Date 对象、时间格式化
- **异步编程**：回调、Promise、async/await
- **DOM 操作**：元素选择、事件处理、DOM 操作

[📚 深入学习核心工具 →](https://developer.mozilla.org/en-US/docs/Learn/JavaScript)

## Phase 3: 高级特性

掌握 JavaScript 的高级编程特性和现代 JavaScript。

- **异步进阶**：Promise 链、async/await、错误处理
- **模块系统**：ES6 模块、CommonJS、动态导入
- **类与继承**：class、extends、super、静态方法
- **代理与反射**：Proxy、Reflect、元编程
- **迭代器与生成器**：迭代器协议、生成器函数、yield
- **Web API**：Fetch API、WebSocket、Web Workers

[📚 深入学习高级特性 →](https://developer.mozilla.org/en-US/docs/Learn/JavaScript)

## Phase 4: 底层原理

深入理解 JavaScript 运行机制和性能优化。

- **JavaScript 引擎**：V8 引擎、执行上下文、调用栈
- **事件循环**：宏任务、微任务、Event Loop
- **内存管理**：垃圾回收、内存泄漏、性能优化
- **性能优化**：性能分析、优化技巧、最佳实践
- **工具链**：npm、webpack、babel、构建工具
- **最佳实践**：代码规范、设计模式、测试

[📚 深入学习底层原理 →](https://developer.mozilla.org/en-US/docs/Learn/JavaScript)

---

💡 **提示**：每个阶段的学习内容都链接到 [JavaScript 官方教程](https://developer.mozilla.org/en-US/docs/Learn/JavaScript)，建议按照顺序系统学习。
