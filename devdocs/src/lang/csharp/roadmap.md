# C# 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 C# 语言的基本语法和面向对象编程核心概念。

- **变量与数据类型**：值类型、引用类型、可空类型
- **控制流**：条件语句、循环、跳转语句
- **面向对象编程**：类与对象、封装、继承、多态
- **接口与抽象类**：接口定义、抽象类、密封类
- **命名空间**：命名空间组织、using 指令
- **属性与索引器**：属性定义、自动属性、索引器

[📚 深入学习语法基础 →](https://learn.microsoft.com/en-us/dotnet/csharp/)

## Phase 2: 核心工具

掌握 .NET 框架中的核心工具类和常用功能。

- **集合类型**：List、Dictionary、HashSet、Queue、Stack
- **LINQ**：查询语法、方法语法、延迟执行
- **异常处理**：try-catch-finally、异常类型、自定义异常
- **文件操作**：文件读写、流操作、异步IO
- **字符串处理**：String、StringBuilder、格式化
- **日期时间**：DateTime、TimeSpan、时区处理

[📚 深入学习核心工具 →](https://learn.microsoft.com/en-us/dotnet/csharp/)

## Phase 3: 高级特性

掌握 C# 的高级编程特性和现代 C# 特性。

- **泛型**：泛型类、泛型方法、约束、协变逆变
- **委托与事件**：委托、事件、Lambda 表达式
- **异步编程**：async/await、Task、异步模式
- **反射**：Type、MethodInfo、属性操作、动态创建
- **特性（Attributes）**：自定义特性、反射特性
- **模式匹配**：is 表达式、switch 表达式、模式匹配（C# 7+）

[📚 深入学习高级特性 →](https://learn.microsoft.com/en-us/dotnet/csharp/)

## Phase 4: 底层原理

深入理解 .NET 运行机制和性能优化。

- **.NET 运行时**：CLR、IL、JIT 编译、程序集
- **内存管理**：垃圾回收、内存分析、性能优化
- **并发编程**：Thread、Task、并行库、并发集合
- **性能优化**：性能分析、优化技巧、最佳实践
- **.NET 生态**：NuGet、包管理、项目结构
- **最佳实践**：代码规范、设计模式、单元测试

[📚 深入学习底层原理 →](https://learn.microsoft.com/en-us/dotnet/csharp/)

---

💡 **提示**：每个阶段的学习内容都链接到 [C# 官方教程](https://learn.microsoft.com/en-us/dotnet/csharp/)，建议按照顺序系统学习。
