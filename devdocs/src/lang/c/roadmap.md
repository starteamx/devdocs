# C 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 C 语言的基本语法和核心概念。

- **变量与数据类型**：基本类型、类型转换、常量
- **控制流**：条件语句、循环、跳转语句
- **函数**：函数定义、参数传递、递归
- **数组与字符串**：数组、字符串、字符数组
- **指针基础**：指针概念、指针运算、指针与数组
- **结构体**：结构体定义、结构体指针、联合体

[📚 深入学习语法基础 →](https://www.learn-c.org/)

## Phase 2: 核心工具

掌握 C 标准库中的核心工具和常用功能。

- **标准 IO**：printf、scanf、文件操作
- **字符串处理**：strlen、strcpy、strcmp、字符串函数
- **内存管理**：malloc、free、calloc、realloc
- **动态内存**：堆内存分配、内存泄漏、内存对齐
- **文件操作**：文件读写、文件指针、二进制文件
- **预处理器**：宏定义、条件编译、头文件

[📚 深入学习核心工具 →](https://www.learn-c.org/)

## Phase 3: 高级特性

掌握 C 语言的高级编程特性和系统编程。

- **指针进阶**：函数指针、指针数组、多级指针
- **数据结构**：链表、栈、队列、树
- **位操作**：位运算、位字段、位掩码
- **动态数据结构**：动态数组、链表实现
- **系统调用**：文件系统、进程、信号
- **多文件编程**：头文件组织、模块化设计

[📚 深入学习高级特性 →](https://www.learn-c.org/)

## Phase 4: 底层原理

深入理解 C 运行机制和系统级编程。

- **内存模型**：栈、堆、静态存储区、内存布局
- **编译与链接**：编译过程、链接器、符号表
- **性能优化**：性能分析、优化技巧、内联汇编
- **嵌入式编程**：寄存器操作、中断处理、硬件接口
- **标准库深入**：标准库实现、POSIX 接口
- **最佳实践**：代码规范、内存安全、可移植性

[📚 深入学习底层原理 →](https://www.learn-c.org/)

---

💡 **提示**：每个阶段的学习内容都链接到 [C 语言学习教程](https://www.learn-c.org/)，建议按照顺序系统学习。
