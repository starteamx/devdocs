# CSS 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 CSS 语言的基本语法和核心概念。

- **CSS 基础**：选择器、属性、值、单位
- **盒模型**：width、height、padding、margin、border
- **布局基础**：display、position、float、clear
- **文本样式**：font、color、text-align、line-height
- **背景与边框**：background、border、border-radius
- **选择器**：元素选择器、类选择器、ID 选择器、伪类

[📚 深入学习语法基础 →](https://developer.mozilla.org/en-US/docs/Learn/CSS)

## Phase 2: 核心工具

掌握 CSS 的核心功能和布局技术。

- **Flexbox**：flex 容器、flex 项目、对齐方式
- **Grid 布局**：grid 容器、grid 项目、网格线
- **响应式设计**：媒体查询、viewport、移动优先
- **动画与过渡**：transition、animation、关键帧
- **变换**：transform、translate、rotate、scale
- **CSS 变量**：自定义属性、var()、作用域

[📚 深入学习核心工具 →](https://developer.mozilla.org/en-US/docs/Learn/CSS)

## Phase 3: 高级特性

掌握 CSS 的高级功能和现代 CSS 特性。

- **CSS 预处理器**：Sass、Less、Stylus、变量与混入
- **CSS 模块化**：BEM、CSS Modules、作用域隔离
- **高级选择器**：属性选择器、伪元素、组合选择器
- **CSS 函数**：calc()、var()、url()、rgb()、hsl()
- **容器查询**：@container、容器单位、响应式组件
- **层叠与继承**：优先级、!important、继承规则

[📚 深入学习高级特性 →](https://developer.mozilla.org/en-US/docs/Learn/CSS)

## Phase 4: 底层原理

深入理解 CSS 渲染机制和性能优化。

- **渲染机制**：渲染树、重排、重绘、合成层
- **性能优化**：CSS 优化、关键 CSS、代码分割
- **浏览器兼容性**：前缀、polyfill、特性检测
- **CSS 架构**：代码组织、命名规范、最佳实践
- **现代 CSS**：CSS 新特性、Houdini、CSS-in-JS
- **最佳实践**：代码规范、性能优化、可维护性

[📚 深入学习底层原理 →](https://developer.mozilla.org/en-US/docs/Learn/CSS)

---

💡 **提示**：每个阶段的学习内容都链接到 [CSS 官方教程](https://developer.mozilla.org/en-US/docs/Learn/CSS)，建议按照顺序系统学习。
