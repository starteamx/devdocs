# Java 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 Java 语言的基本语法和面向对象编程核心概念。

- **变量与数据类型**：基本类型、引用类型、类型转换
- **控制流**：条件语句（if/else、switch）、循环（for、while、do-while）
- **面向对象编程**：类与对象、封装、继承、多态
- **接口与抽象类**：接口定义与实现、抽象类使用
- **包与模块**：包的创建与使用、模块系统（Java 9+）
- **数字与字符串**：数值处理、字符串操作、格式化

[📚 深入学习语法基础 →](https://dev.java/learn/)

## Phase 2: 核心工具

掌握 Java 标准库中的核心工具类和框架。

- **集合框架**：List、Set、Map、Queue 的使用与选择
- **IO / NIO**：文件读写、流操作、NIO 非阻塞 IO
- **异常处理**：try-catch-finally、自定义异常、异常最佳实践
- **日期时间 API**：LocalDate、LocalTime、DateTimeFormatter（Java 8+）
- **正则表达式**：Pattern、Matcher 的使用
- **记录类型（Records）**：不可变数据建模（Java 14+）

[📚 深入学习核心工具 →](https://dev.java/learn/)

## Phase 3: 高级特性

掌握 Java 的高级编程特性和现代编程范式。

- **并发编程**：Thread、Runnable、Executor 框架、虚拟线程（Java 21+）
- **Stream API**：流式处理、函数式编程、并行流
- **Lambda 表达式**：函数式接口、方法引用
- **泛型**：泛型类、泛型方法、通配符、类型擦除
- **反射与注解**：Class、Method、Field 操作、自定义注解
- **模式匹配**：instanceof 模式匹配、switch 表达式（Java 14+）

[📚 深入学习高级特性 →](https://dev.java/learn/)

## Phase 4: 底层原理

深入理解 JVM 运行机制和性能优化。

- **JVM 虚拟机**：内存模型、堆栈结构、方法区
- **垃圾回收机制**：GC 算法、GC 调优、内存分析
- **类加载机制**：类加载过程、类加载器、双亲委派模型
- **性能调优工具**：JVM 监控工具、JFR（Java Flight Recorder）、性能分析
- **模块系统**：模块化开发、jlink 工具、自定义运行时镜像
- **本地接口**：JNI、FFM API（Foreign Function & Memory API，Java 19+）

[📚 深入学习底层原理 →](https://dev.java/learn/)
