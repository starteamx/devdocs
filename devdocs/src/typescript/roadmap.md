# TypeScript 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 TypeScript 语言的基本语法和类型系统。

- **类型基础**：基本类型、联合类型、字面量类型
- **变量与函数**：类型注解、函数类型、可选参数
- **接口**：接口定义、可选属性、只读属性
- **类**：类定义、继承、访问修饰符
- **类型推断**：类型推断、类型断言、类型守卫
- **枚举**：数字枚举、字符串枚举、常量枚举

[📚 深入学习语法基础 →](https://www.typescriptlang.org/docs/handbook/intro.html)

## Phase 2: 核心工具

掌握 TypeScript 的核心类型系统和常用特性。

- **泛型**：泛型函数、泛型接口、泛型类、约束
- **高级类型**：交叉类型、联合类型、映射类型
- **工具类型**：Partial、Required、Pick、Omit、Record
- **模块系统**：ES 模块、命名空间、模块解析
- **装饰器**：类装饰器、方法装饰器、属性装饰器
- **类型操作**：keyof、typeof、infer、条件类型

[📚 深入学习核心工具 →](https://www.typescriptlang.org/docs/handbook/intro.html)

## Phase 3: 高级特性

掌握 TypeScript 的高级编程特性和类型系统。

- **类型系统进阶**：条件类型、分布式条件类型、模板字面量类型
- **声明合并**：接口合并、命名空间合并、模块扩充
- **类型编程**：类型工具、类型体操、复杂类型推导
- **工程化**：tsconfig.json、编译选项、项目配置
- **类型声明**：.d.ts 文件、声明文件、第三方类型
- **最佳实践**：类型设计、代码组织、性能优化

[📚 深入学习高级特性 →](https://www.typescriptlang.org/docs/handbook/intro.html)

## Phase 4: 底层原理

深入理解 TypeScript 编译机制和类型系统。

- **编译器**：编译过程、类型检查、代码生成
- **类型系统**：结构化类型、名义类型、类型兼容性
- **性能优化**：编译性能、类型检查性能、增量编译
- **工具链**：tsc、ts-node、类型检查工具
- **项目配置**：tsconfig.json 详解、路径映射、项目引用
- **最佳实践**：类型安全、代码规范、团队协作

[📚 深入学习底层原理 →](https://www.typescriptlang.org/docs/handbook/intro.html)

---

💡 **提示**：每个阶段的学习内容都链接到 [TypeScript 官方教程](https://www.typescriptlang.org/docs/handbook/intro.html)，建议按照顺序系统学习。
