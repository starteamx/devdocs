# C++ 学习路线图

> 基于官方教程的系统化学习路径，按照"基础 → 核心 → 进阶 → 底层"四个阶段循序渐进。

## Phase 1: 语法基础

掌握 C++ 语言的基本语法和核心概念。

- **变量与数据类型**：基本类型、引用、指针
- **控制流**：条件语句、循环、跳转语句
- **函数**：函数定义、重载、默认参数
- **数组与字符串**：数组、字符串、C风格字符串
- **结构体与类**：结构体、类、访问控制
- **命名空间**：命名空间定义、using 声明

[📚 深入学习语法基础 →](https://www.learncpp.com/)

## Phase 2: 核心工具

掌握 C++ 标准库中的核心工具和常用特性。

- **STL 容器**：vector、list、map、set、unordered_map
- **STL 算法**：排序、查找、变换、迭代器
- **IO 流**：文件操作、字符串流、格式化输出
- **异常处理**：try-catch、异常类型、RAII
- **智能指针**：unique_ptr、shared_ptr、weak_ptr
- **模板基础**：函数模板、类模板、模板特化

[📚 深入学习核心工具 →](https://www.learncpp.com/)

## Phase 3: 高级特性

掌握 C++ 的高级编程特性和现代 C++ 特性。

- **面向对象**：继承、多态、虚函数、抽象类
- **运算符重载**：运算符重载规则、常用运算符
- **模板进阶**：模板元编程、SFINAE、concepts（C++20）
- **Lambda 表达式**：Lambda 语法、捕获、应用场景
- **移动语义**：右值引用、移动构造、完美转发
- **并发编程**：thread、mutex、condition_variable、async

[📚 深入学习高级特性 →](https://www.learncpp.com/)

## Phase 4: 底层原理

深入理解 C++ 运行机制和性能优化。

- **内存管理**：堆栈内存、内存对齐、内存池
- **编译与链接**：编译过程、链接器、静态/动态库
- **性能优化**：性能分析、内联、优化技巧
- **RAII 原则**：资源管理、异常安全
- **现代 C++ 特性**：C++11/14/17/20 新特性
- **最佳实践**：代码规范、设计模式、测试

[📚 深入学习底层原理 →](https://www.learncpp.com/)

---

💡 **提示**：每个阶段的学习内容都链接到 [C++ 学习教程](https://www.learncpp.com/)，建议按照顺序系统学习。
